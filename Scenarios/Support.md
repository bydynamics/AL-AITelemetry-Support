# Support

## Getting help

- **Email:** [support@bydynamics.com](mailto:support@bydynamics.com)
- **GitHub Issues:** [AL-AITelemetry-Support](https://github.com/bydynamics/AL-AITelemetry-Support/issues)

## Reporting a bug

When reporting a bug, please include:

1. The Business Central version (e.g., *28.1*).
2. The AI Telemetry app version (found in **Extension Management**).
3. Steps to reproduce the issue.
4. Any error messages or screenshots.
5. The query that caused the problem (if applicable).

## Feature requests

We welcome feature suggestions! Open a GitHub issue with the label
**enhancement** and describe your use case.

## Release notes

Release notes for each version are published on the
[GitHub Releases](https://github.com/bydynamics/AL-AITelemetry-Support/releases)
page.
