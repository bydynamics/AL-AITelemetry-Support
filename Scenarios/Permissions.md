# Permissions

## Permission sets

AI Telemetry ships with the following permission sets:

| Permission Set | Description |
|----------------|-------------|
| **AI Telemetry - Admin** | Full access: setup, query management, and execution |
| **AI Telemetry - User** | Execute queries and view results (no setup access) |

## Assigning permissions

1. Open **Permission Sets** in Business Central.
2. Search for *AI Telemetry*.
3. Assign the appropriate permission set to users or user groups.

## Minimum requirements

- To configure the connection: **AI Telemetry - Admin**.
- To run queries and view results: **AI Telemetry - User**.
- To use Copilot features: **AI Telemetry - User** (Copilot capabilities must
  also be enabled at environment level).
