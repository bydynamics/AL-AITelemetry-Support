### AI Telemetry

AI Telemetry lets you query Azure Application Insights telemetry directly from
Microsoft Dynamics 365 Business Central using natural language. Leveraging
Copilot, the app generates intelligent KQL queries with support for query
management, parameterized queries, and full Business Central telemetry schema
awareness.
